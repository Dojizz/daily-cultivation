# Backlog

后续功能候选，按优先级从高到低排列（内部不分先后）。

## 待评估 & 设计

### 日课回顾增强

- 特质每日自评：为已完成的特质记录 1–5 分的自我评分，并在回顾日历中展示评分强度。
- 该功能需要为 `practice_records` 新增持久化字段及 Room migration；实施前单独完成数据模型和交互设计。

### 日记

- 每日记录，纯文本为主
- 自动汇总当日其他模块数据（完成了哪些任务、当日日课评分等），用户只需写感受
- 日记不是独立孤岛，是"今日视图"的一部分

### 备忘录 / 检查清单

- 场景化模板：
  - 长途旅行前带什么
  - 短途出差带什么
  - 回家带什么
  - 出门前检查项（关热水器、关洗衣机盖、植物浇水等）
- 每项可勾选，每次使用前重置

### 财务预算管理

- 月度收入设定
- 多分类预算（餐饮、交通、娱乐、学习等）
- 支出记录，实时显示各类别剩余额度
- 月度汇总
